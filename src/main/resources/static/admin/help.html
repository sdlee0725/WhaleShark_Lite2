• 사용안내
                    사용순서:
                    
                    초기 접속 계정 : admin/admin
                    
                    1. 사용자및권한관리> 사용자그룹관리 : 사용자에게 지정할 그룹을 관리한다.
                    
                    2. 사용자및권한관리> 사용자관리 : 사용자 계정을 관리한다.
                    
                    3. 데이터 통합 : 메타 관리 및 데이터 수집.
                    
                    4. 데이터 브라우즈 : 통합 데이터 조회 및 기본 처리.
                    
                    5. 분석 : 데이터분석
                    
                    6. 시각화 : 데이터 분석 결과 또는 수집 데이터의 visualization.
                    
                    7. 시스템 모니터링 : 자원의 상태를 모니터링 및 사용법.
